## Introduction
이 챗봇 서비스는 사용자가 작성한 일기, 챗봇 간의 대화 내용 및 이를 토대로 나온 그날의 상태 분석결과 등을 바탕으로 컨텐츠를 추천해주거나 조언을 해주는 등 사용자의 심리를 안정시켜주는 멘탈케어 서비스입니다.
